# Portfolio — Christian Dibi

Portfolio personnel de **Dibi Atta Joseph Christian Marie** — Developpeur Python & Odoo | DevOps.

Site statique (HTML / CSS / JS), sans dependance ni build, publie via **GitHub Pages**.

## Structure

```
PorteFolio/
├── index.html          # Page unique
├── css/style.css       # Styles + theme clair/sombre
├── js/script.js        # Theme toggle, menu mobile, animations
├── assets/CV_*.pdf     # CV telechargeable
└── README.md
```

## Lancer en local

Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
python -m http.server 8000
# puis http://localhost:8000
```

## Publication GitHub Pages

Le site est deploye depuis la branche `main`, dossier racine (`/root`).
URL : https://dibichristian.github.io/PorteFolio/
